# FlashBasket 🛒

FlashBasket is a Mini Grocery Delivery Android App inspired by Blinkit.

It allows users to browse products, add items to cart, place orders, and make payments using COD or Online UPI.

---

# Features

## 1. Login / OTP
- Mobile number login
- Fake OTP verification
- Navigation to Home Screen

## 2. Home Screen
- Product listing
- Product images
- Search functionality
- Category filters
- Add to cart
- Wishlist support

## 3. Cart Screen
- Increase quantity
- Decrease quantity
- Remove items automatically
- GST calculation
- Coupon code support
- Free delivery above ₹199

## 4. Checkout Screen
- Delivery address input
- Cash on Delivery (COD)
- Online UPI Payment

## 5. Order Success
- Success message
- Order ID
- Estimated delivery time
- Order tracking

## Bonus Features
- Product Details Screen
- Wishlist Screen
- Profile Screen
- Bottom Navigation
- Splash Screen
